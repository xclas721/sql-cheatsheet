# SQL 語法速查表

一個互動式的 SQL 語法速查表，提供常用 SQL 語法的示例和說明。

## 功能特點

- 分類展示 SQL 語法
- 互動式示例
- 代碼高亮顯示
- 一鍵複製功能
- 響應式設計，支援手機瀏覽

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 構建生產版本
npm run build
```

## 部署

本項目使用 GitHub Pages 進行部署。每次推送到 main 分支時，GitHub Actions 會自動構建並部署到 gh-pages 分支。

## 技術棧

- Vue 3
- Vite
- Prism.js
- GitHub Pages
